# Get Started

A Step-by-step Guide to Using Moja UI

## Setup Guide

1. ### Installation

To use Moja UI in your project, run one of the following commands in your terminal:

<Tabs className="custom-tabs">
	<TabItem value="npm" label="npm">
		``` npm i @moja-ui/react ```
	</TabItem>
	<TabItem value="pnpm" label="pnpm">
		``` pnpm add @moja-ui/react```
	</TabItem>
	<TabItem value="yarn" label="yarn" default>
		``` yarn add @moja-ui/react ```
	</TabItem>
	<TabItem value="bun" label="bun">
		```bash bun add @moja-ui/react ```
	</TabItem>
</Tabs>

1. ### Add a component

In this guide, we will be adding a Button component. Copy the following code to your project.

```js
import { Button, type ButtonProps } from "@mojaui/button";

export const Demo = (props: ButtonProps) => {
	return (
		<Button {...props}>
			Button
			<ArrowRightIcon />
		</Button>
	);
};
```
